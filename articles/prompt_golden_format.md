# 内容

プロンプトで黄金フォーマットを使った思考整理をする。

# プロンプト

```text
これから黄金フォーマットを使った思考整理を行います。
あなたは私が効率よく思考整理するお手伝いをしてください。

黄金フォーマットには次の4つの領域があります。

領域1：背景、前提
領域2：課題、仮説
領域3：手段、行動
領域4：効果、結論

それぞれの領域は次のような事柄が当てはまります。

領域1
・それまでの状態
・今まで進行していたもの
・前提条件

領域2
・現在直面している問題や課題
・課題と捉える理由
・課題に対する仮説

領域3
・どのように問題を解決しようとしているのか
・その問題解決手段を選んだ理由
・そのアプローチの意味や意義

領域4
・得られる結果から何が言えるのか
・なぜそれが言えるのか
・どのような変化が起こるのか

それぞれの領域は次のような関係にあります。

領域1と領域2 の内容は、現状と課題の逆説関係であり相反し合う内容である。
領域1と領域3 の内容は、手段には現状との差分のみを記載する。
領域3と領域4の内容は、原因と結果の順接関係である。
領域2と領域4の内容は、問題/課題に対して解決/達成の裏返しの関係である。

あなたは、私に対して領域1~4の内容を順番に問いかけてください。
あなたの問いかけに私が答えます。

領域1〜4すべての問いが終わったら、あなたは私が答えた内容が黄金フォーマットに則っているかをレビューしてください。
レビューは次の観点でお願いします。

・各領域への答えが、その領域の内容として適切であること
・各領域の関係性が適切であること

とくに関係性は重点的にレビューしてください。
もし、より良いレビューのために情報が不足している場合は聞き返してください。

それでは、問いかけを始めてください。
```
